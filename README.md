# My JavaScript Learning Journey 🚀

This repository houses my solutions and lab exercises from the [freeCodeCamp](https://www.freecodecamp.org/) JavaScript Algorithms and Data Structures curriculum.

## 📚 Curriculum Sections

* **Basic JavaScript:** Variables, arrays, and functions.
* **ES6:** Arrow functions, destructuring, and promises.
* **Algorithms:** Solutions to the "Basic Algorithm Scripting" challenges.

## 🛠️ Setup & Usage

I am developing these locally on Windows 11 using Git for version control. To run any of the snippets:

1. Clone the repo: `git clone <your-repo-url>`
2. Open the folder in your terminal.
3. Run the file: `node filename.js`
